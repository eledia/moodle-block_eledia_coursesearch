# So stellen Sie das Plugin für alle Benutzer bereit
## Boost Union Theme
Da es sich bei dem Plugin um einen *block*-Plugin handelt, wird empfohlen, es nur auf dem Dashboard anzuzeigen.  
Um das Plugin allen Benutzern auf dem Dashboard bereitzustellen:
- Gehen Sie zu *Website-Administration* **&arr;** *Darstellung* **&arr;** *Dashboard für alle anpassen*  
- Aktivieren Sie den *Bearbeitungsmodus*
- Klicken Sie auf *Block hinzufügen*
- Fügen Sie das Plugin hinzu
- Deaktivieren Sie den *Bearbeitungsmodus*
- Klicken Sie auf *Dashboard für alle Nutzer/innen übernehmen*
Jetzt ist die Kurssuche für alle Benutzer sichtbar.
